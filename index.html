Limite de Navigateur

Un script Python Discord ne peut pas tourner en arrière-plan d'une simple page web HTML (blocage CORS/Sécurité). Tu dois l'exécuter sur un vrai environnement.

Comment Lancer :

Option A : Sur ton PC (Local)

  1. Clique sur "Télécharger .py" en haut de cette page.
  2. Ouvre un Terminal (CMD ou PowerShell).
  3. Installe la librairie requise : pip install discord.py
  4. Démarre ton bot : python bot.py

Option B : En ligne (Replit)

  1. Copie le code python via le bouton "Copier".
  2. Va sur Replit.com et crée un projet Python.
  3. Colle le code dans `main.py`.
  4. Clique sur Run ▶
